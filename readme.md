#  My To-Do List App

A cute and playful to-do list web app built with HTML, CSS and JavaScript!

## Features
- Add tasks with date and time
- Mark tasks as done
- Delete tasks
- Filter by All / Active / Completed
- Task counter
- Celebration GIF when all tasks are done
- ⚠️ Reminder for incomplete tasks from previous day
- Auto deletes completed tasks next day
- Saves tasks even after refresh

## Built With
- HTML
- CSS
- JavaScript
- localStorage
